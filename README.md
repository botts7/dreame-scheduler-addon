# Dreame Scheduler — Home Assistant Add-on

Companion configuration panel for the
[Dreame Scheduler integration](https://github.com/botts7/dreame-scheduler).
Install the integration first, then add this repository.

## Install

1. In Home Assistant: **Settings → Add-ons → Add-on Store → ⋮ → Repositories**,
   add `https://github.com/botts7/dreame-scheduler-addon`.
2. Install **Dreame Scheduler**, start it, and open it from the sidebar.

Prebuilt multi-arch images (amd64 / aarch64 / armv7) are published to GHCR, so
installation doesn't compile anything locally.

## What it does

A GUI (behind ingress) for the integration: presence, cleaning window, guards,
per-room schedules, a report view, ready-to-paste dashboard cards, and the
optional Floor Plan Studio. See the add-on's **Documentation** tab for details.
